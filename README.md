# LaunchFast Landing Page

Simple landing page for Amazon FBA Product Research Tool.

## Deploy to Railway

1. Connect this repository to Railway
2. Railway will auto-detect Next.js
3. Set environment variable: `PORT=3000` (optional)
4. Deploy!

## Local Development

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Build

```bash
npm run build
npm start
```
